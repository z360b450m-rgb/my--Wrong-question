## 1. 错题本管理

### 1.1 多错题本支持
- 可创建多个错题本（Notebook），每个错题本独立管理一套错题
- 错题本包含：**名称**、**简介** 两个元数据字段
- 每个错题本在主页以表格行样式展示，左侧侧边栏列出所有错题本名称

### 1.2 错题本操作
- **新建错题本**：点击"新建错题本"按钮，填写名称和简介，支持回车快捷确认
- **进入错题本**：点击错题本名称进入对应错题本的工作区
- **返回菜单**：在工作区侧边栏顶部点击"返回"按钮，回到错题本选择页面
- **拖拽排序**：在侧边栏中拖拽错题本名称可重排顺序（自定义排序持久化存储）
- **数据隔离**：每个错题本的条目完全独立，切换错题本时重新加载数据
- **搜索过滤**：支持按名称搜索过滤错题本

### 1.3 随机励志名言
- 每次进入错题本选择页面，从 16 条中文名言库中随机展示一条

## 2. 错题条目管理

### 2.1 创建错题
- 通过工具栏"新建错题"按钮（或 Ctrl+N）创建
- 在侧边栏学科标签上点击 **"+" 按钮**，可创建带有该学科标签的新错题
- 新错题自动生成标题（格式："无题目 (N)"，N 自增）
- 新建后自动选中并进入编辑状态

### 2.2 编辑错题
每条错题包含以下可编辑字段：

| 字段 | 说明 | 输入方式 |
|------|------|----------|
| 标题 | 自动生成，可双击侧边栏条目重命名 | 文本输入 |
| 学科 | 如"言语""数学""判断"等 | 文本输入（100px 宽度） |
| 来源 | 如"月考""模拟卷"等 | 文本输入（120px 宽度） |
| 标签 | 逗号分隔的多标签 | 文本输入 |
| 题目 | 支持 Markdown 和图片粘贴 | 富文本编辑（contenteditable） |
| 错误答案 | 自己写的错误答案 | 富文本编辑 |
| 正确答案 | 标准正确答案 | 富文本编辑，默认隐藏 |

### 2.3 标题管理
- 新建时自动生成占位标题
- 保存时，如果题目有内容且标题仍为占位标题，自动截取题目前 40 个字符作为标题
- 在侧边栏条目列表中**双击标题**可直接重命名（内联编辑），Enter 确认，Esc 取消，点击其他地方自动保存

### 2.4 保存与脏状态
- 编辑任何字段后进入"脏"状态（isDirty），工具栏保存按钮高亮并显示红点
- **手动保存**：点击保存按钮或 Ctrl+S，调用 `saveEntry()` 持久化到数据库
- **自动快照**：脏状态下每 1 秒自动保存一次快照（用于崩溃恢复）
- **未保存提示**：当有未保存更改时切换条目/新建/进入复习，弹出"未保存的更改"对话框：
  - **保存**：先保存再继续操作
  - **不保存**：丢弃修改，从数据库恢复原始数据
  - **取消**：取消操作，留在当前页面

### 2.5 删除错题
- 单条删除：工具栏菜单 → 删除 → 弹出确认对话框
- 确认后同时删除条目数据和关联快照

## 3. 编辑器功能

### 3.1 题目编辑器
- **富文本编辑**：使用 contenteditable div，支持 Markdown 语法渲染
- **图片粘贴**：Ctrl+V 直接粘贴剪贴板中的图片，自动转为 base64 嵌入
- **图片拖放**：拖拽图片文件到编辑区域自动插入
- **拍照插入**：面板标题栏右上角悬浮工具栏，支持摄像头拍照
- **文件导入**：面板标题栏右上角悬浮工具栏，支持从本地选择图片文件
- **桌面截图**（仅 Electron）：截取桌面窗口或屏幕并插入到编辑器
- **截图裁剪**：截图后可交互式裁剪后再插入
- 工具栏按钮仅在鼠标悬停时显示（`opacity-0 group-hover:opacity-100`）

### 3.2 答案面板（AnswerPanel）
- 错误答案和正确答案各为独立的富文本编辑面板
- 错误答案面板始终可见
- 正确答案面板在编辑模式下默认隐藏（`answersHidden`），点击工具栏"显示正确答案"或 Ctrl+R 切换显示
- 图片粘贴、拖放、拍照、文件导入功能同题目编辑器
- 面板内支持 contenteditable 编辑

### 3.3 面板布局
- **题目区 + 答案区**上下排列
- 中间有**可拖拽调节手柄**，拖动改变题目区和答案区的高度比例
- 答案区左右分栏（错误答案 | 正确答案），中间有**可拖拽调节手柄**改变左右宽度比例
- 窄屏（<500px）时答案区自动切换为上下堆叠布局

### 3.4 Markdown 渲染
- 题目和答案区域支持 Markdown 语法显示（通过 CSS `.md-content` 类）
- 占位提示文字："在此输入题目内容…"

## 4. 学科与标签系统

### 4.1 学科（Subject）
- 每个错题可关联一个学科（如：言语、数学、判断推理、资料分析等）
- 侧边栏展示学科分布芯片，显示每个学科的错题数量
- 学科芯片由两部分组成：
  - **左侧主体**：点击筛选该学科的错题，选中高亮
  - **右侧"+"按钮**：在该学科下快速新建错题（自动带上学科标签）
- "全部"芯片：显示所有错题
- "未分类"芯片：显示没有学科标签的错题

### 4.2 标签（Tags）
- 每个错题可关联多个标签，用逗号分隔（如：易错, 重点, 公式）
- 侧边栏展示标签分布，点击筛选
- 点击已选中的标签可取消筛选（toggle 行为）
- 选中学科后自动清除标签筛选

## 5. 筛选与排序

### 5.1 多维度筛选
- **学科筛选**：选择特定学科或未分类
- **标签筛选**：选择特定标签
- **掌握程度筛选**：按 6 个等级筛选（未复习 / 未掌握 / 一次复习 / 二次复习 / 三次复习 / 已掌握），再次点击取消筛选
- **关键词搜索**：全文搜索题目、错误答案、正确答案、学科、来源字段

### 5.2 排序方式
| 排序方式 | 说明 |
|----------|------|
| 更新时间 | 按最近修改时间排序（默认），支持升序/降序 |
| 创建时间 | 按创建时间排序，支持升序/降序 |
| 学科 | 按学科名称拼音排序 |
| 标题 | 按标题拼音排序 |
| 自定义 | 手动拖拽排序（持久化），支持升序/降序 |
| 乱序 | 随机打乱顺序（每次点击重新随机） |

- 点击排序标签切换排序方式，再次点击切换升序/降序

## 6. 间隔重复复习系统 (SRS)

### 6.1 算法原理
基于 SM-2（SuperMemo 2）改进的自定义间隔重复算法：

- **复习间隔计算**：
  - 第 0 次通过（passes=0）：首次复习天数（默认 1 天）
  - 第 1 次通过（passes=1）：已掌握天数（默认 3 天）
  - 第 2 次通过（passes=2）：已掌握天数 × 增长系数
  - 第 3 次通过（passes=3）：当前间隔 × 增长系数（不超过最大间隔）
  - 第 N 次通过：当前间隔 × 增长系数（不超过最大间隔）

### 6.2 掌握程度（Mastery Level）
| 等级 | 标签 | 颜色 | 条件 |
|------|------|------|------|
| -1 | 未复习 | 灰色 `#9ca3af` | 从未复习过 |
| 0 | 未掌握 | 红色 `#ef4444` | 刚复习但被评为"遗忘" |
| 1 | 一次复习 | 橙色 `#f59e0b` | 连续通过 1 次 |
| 2 | 二次复习 | 靛蓝 `#6366f1` | 连续通过 2 次 |
| 3 | 三次复习 | 紫色 `#8b5cf6` | 连续通过 3 次 |
| 4 | 已掌握 | 绿色 `#22c55e` | 连续通过 4+ 次 |

### 6.3 评级系统
每次复习后给出三个等级的评分：

| 评级 | 快捷键 | 效果 |
|------|--------|------|
| 完全未掌握 (forgot) | 1 | 连续通过次数归零，间隔重置为未掌握天数，掌握等级降为 0 |
| 不熟练 (unfamiliar) | 2 | 连续通过次数不变（首次则为 0），间隔按未掌握天数或当前等级计算，掌握等级不变 |
| 已掌握 (mastered) | 3 | 连续通过次数 +1，间隔按新等级计算，掌握等级升级 |

### 6.4 复习调整设置（可配置）
| 参数 | 默认值 | 范围 | 说明 |
|------|--------|------|------|
| 首次复习天数 | 1 天 | 0-365 | 新条目首次复习间隔 |
| 未掌握天数 | 1 天 | 1-365 | 被评为"遗忘"后的重新复习间隔 |
| 已掌握天数 | 3 天 | 1-365 | 连续通过 2 次后的基础间隔 |
| 增长系数 | 2.5× | 1.1-5.0 | 每次连续通过后间隔的放大倍数 |
| 最大间隔 | 180 天 | 7-730 | 复习间隔的上限 |

- 设置保存在 `localStorage`，键名 `cuoti-review-settings`
- 修改间隔参数后，所有现有条目的间隔和下次复习日期**自动立即重算**

## 7. 复习模式

### 7.1 启动复习
- 在编辑模式下点击"复习模式"按钮或 Ctrl+R
- 如果当前有待复习条目（`dueCount > 0`），自动开始复习
- 如果今日已全部复习，提示"今日复习已完成"，但仍可点击强制复习全部条目
- 有未保存更改时，先弹出保存对话框

### 7.2 复习流程
1. 所有待复习条目随机打乱后形成复习队列
2. 先展示题目（含学科标签），**同时启动计时器**
3. 错误答案默认展开显示，正确答案默认折叠（点击展开）
4. 点击正确答案面板查看答案 → 进入评分阶段
5. 可选填写**复习批注**（纯文本，保存到错误答案字段末尾，以分隔线隔开）
6. 选择评级（1/2/3）→ 算法更新间隔和掌握等级，记录复习日志
7. 自动进入下一题，直到队列结束
8. 复习结束展示**会话总结**

### 7.3 复习进度指示
- 顶部进度条（百分比）+ 剩余题目数 + 已用时间
- 显示今日已复习数量

### 7.4 会话总结
- 复习完成后展示全屏总结页面
- 列出本次复习的每条错题：序号、掌握等级色点、题目预览、学科、新掌握等级标签、用时、评分
- 总用时统计
- "返回编辑"按钮

### 7.5 中途退出
- 点击"退出复习"按钮可在任意时刻退出，回到编辑模式

### 7.6 复习日志
- 每次复习行为记录到 ReviewLog（含条目 ID、时间戳、评分）
- 每条错题的编辑页面可展开查看**复习历史时间线**（横向滚动，显示每次复习的日期、评分颜色和标签）

## 8. 画笔批注

### 8.1 启用/关闭
- 工具栏"画笔"按钮切换绘图模式
- 开启后工具栏按钮高亮（琥珀色），笔记编辑器的题目区域变为只读
- 关闭后画笔工具栏消失，恢复编辑

### 8.2 绘图工具
- **画笔（Pen）**：6 种颜色可选（红、橙、黄、绿、蓝、紫），线宽 3px，带阴影效果，使用二次贝塞尔曲线平滑
- **橡皮（Eraser）**：线宽 24px，使用 `destination-out` 合成模式擦除

### 8.3 浮动工具栏
- 画笔开启后底部居中显示浮动工具栏
- 包含：笔/橡皮切换、6 种颜色选择、撤销/重做、清除按钮

### 8.4 撤销/重做
- 支持 50 步历史记录（基于 DataURL 快照）
- 快捷键：Ctrl+Z 撤销，Ctrl+Y 重做
- 每次笔画开始时自动保存快照

### 8.5 绘图存储
- 每个条目的绘图数据（DataURL）存储在内存 Map 中（`drawingStore`）
- 切换条目时自动保存当前绘图、加载目标条目绘图
- 保存条目时绘图数据随条目一起持久化到数据库（`entry.drawing` 字段）

### 8.6 画布技术细节
- Canvas 元素覆盖在题目区域上方（`position: absolute; z-index: 10`）
- 支持高 DPI 屏幕（`devicePixelRatio` 缩放）
- ResizeObserver 自动响应容器尺寸变化
- 支持鼠标和触摸事件

## 9. 批量操作

### 9.1 选择机制
- 每个条目左侧有复选框，点击切换选中状态
- **Ctrl+点击**条目切换选中
- **Shift+点击**复选框或条目进行范围选择（从上次选中的条目到当前条目）
- 全选按钮：点击全选当前筛选结果，再次点击取消全选
- 选择后侧边栏顶部显示"已选 N"

### 9.2 批量操作菜单
选中条目后侧边栏顶部出现批量操作下拉菜单：

| 操作 | 说明 |
|------|------|
| 全选/取消全选 | 切换当前筛选结果的全选状态 |
| 批量标签 | 输入逗号分隔的标签，为所有选中条目添加标签（不覆盖已有标签） |
| 批量导出 | 将选中的条目导出为 JSON 文件下载 |
| 批量删除 | 弹出确认对话框后删除所有选中条目（不可撤销） |

### 9.3 批量标签
- 输入框支持逗号/中文逗号分隔多标签
- Enter 确认，Esc 取消
- 新增标签与条目的现有标签合并（不去重覆盖）

## 10. 数据导入导出

### 10.1 导出 JSON
- 导出当前错题本的全部条目为 JSON 文件
- 文件名格式：`错题本_YYYY-MM-DD_HH-MM.json`
- 通过浏览器的 download 机制触发下载

### 10.2 导入 JSON
- 选择 `.json` 文件导入
- 校验：必须是数组、每条必须有 `id` 和 `question` 字段
- 每个导入条目自动生成新 ID（避免与现有条目 ID 冲突）
- 每条导入的条目关联到当前错题本
- 导入后重新加载条目列表
- 显示导入结果：成功 N 条，跳过 N 条无效

### 10.3 导出 PDF（打印）
- 按学科分组生成 HTML 页面，包含所有错题的题目、错误答案、正确答案
- 使用 `window.print()` 触发浏览器打印对话框
- 输出包含：导出日期、总题数、学科分组、每题的来源、标签、题目、答案
- CSS 打印样式：`page-break-inside: avoid` 防止卡片跨页断裂

### 10.4 批量导出
- 选中多条错题后，通过侧边栏批量操作菜单导出选中条目为 JSON

## 12. 设置面板

### 12.1 深色模式
- 切换按钮（iOS 风格滑动开关）
- 设置持久化到 `localStorage`（键名 `cuoti-dark`）
- 默认跟随系统主题偏好（`prefers-color-scheme`）

### 12.2 复习设置（SRS 参数）
- 可展开/折叠的复习参数配置区
- 5 个参数各有独立的数字输入框
- 每个参数旁显示建议默认值
- "保存复习设置"按钮将参数持久化到 `localStorage`
- 修改后所有条目间隔自动立即重算

### 12.3 数据目录（仅 Electron 桌面端）
- 点击"更改保存目录"调用 Electron API 修改本地文件存储位置
- 浏览器端不可用，按钮置灰显示"仅桌面端可用"

### 12.4 打开/关闭
- 左下角齿轮按钮打开设置面板
- 右侧滑出面板（320px 宽）

## 14. 数据持久化

### 14.1 双存储后端
- **Electron 桌面端**：使用文件系统存储（通过 `electronAPI` 桥接），数据保存在本地 JSON 文件中
- **浏览器端**：使用 IndexedDB（数据库名 `CuotiDB`，版本 5）

### 14.2 数据表结构
| Store 名 | 键 | 索引 | 说明 |
|----------|-----|------|------|
| `entries` | `id` | `updatedAt`, `subject` | 错题条目 |
| `snapshots` | `entryId` | — | 崩溃恢复快照 |
| `reviewLogs` | `id` | `entryId` | 复习日志 |
| `notebooks` | `id` | — | 错题本 |

### 14.3 数据迁移
- 首次在 Electron 环境启动时，自动检测文件存储中是否有数据
- 如果没有，从 IndexedDB 迁移所有数据到文件存储
- 迁移完成后显示"已迁移 N 条错题到本地文件"

### 14.4 浏览器存储
- 复习设置：`localStorage` 键 `cuoti-review-settings`
- 深色模式：`localStorage` 键 `cuoti-dark`

## 15. 崩溃恢复

### 15.1 定期快照
- 条目处于脏状态时，每 1 秒自动保存一次快照到 `snapshots` 表
- 快照包含完整的条目数据（`entryId` + `data` + `savedAt`）

### 15.2 页面关闭保护
- `beforeunload` 事件触发时立即保存一次快照

### 15.3 启动恢复
- 应用启动时检查是否有未清理的快照
- 如果快照对应的条目已存在 → 用快照数据覆盖（恢复未保存的编辑）
- 如果快照对应的条目不存在 → 恢复整个条目（恢复因崩溃丢失的新建条目）
- 恢复后清除所有快照，显示"已恢复 N 条未保存的内容"

## 16. 深色模式

- 支持亮色/暗色主题切换
- Tailwind CSS `dark:` 前缀实现主题样式
- 切换时给 `<html>` 和 `<body>` 同时添加/移除 `dark` 类
- 设置持久化，启动时自动加载
- 默认跟随系统主题偏好（无本地设置时）

## 17. 响应式布局

### 17.1 答案面板自适应
- 使用 ResizeObserver 监听答案行宽度
- 容器宽度 < 500px 时，错误答案和正确答案由左右排列切换为上下堆叠
- 窄屏模式下隐藏垂直分隔手柄

### 17.2 可拖拽面板
- 题目区与答案区之间的水平拖拽手柄（调整高度比例）
- 错误答案与正确答案之间的垂直拖拽手柄（调整宽度比例）
- 拖拽时手柄高亮，`body` 添加 `resizing` 类以防止文本选中

### 17.3 画布自适应
- Canvas 尺寸使用 `devicePixelRatio` 适配高 DPI 屏幕
- ResizeObserver 自动调整画布尺寸匹配容器
- 窗口 resize 时重新计算画布尺寸

## 18. UI/UX 细节

### 18.1 列表条目展示
- 每个条目显示：复选框、掌握程度色点、拖拽手柄（自定义排序时）、标题、下次复习标签、学科标签、更新时间、来源
- 当前选中的条目背景高亮（靛蓝色）+ 左侧边框强调
- 被选中的条目显示紫色背景（区别于当前激活条目）
- 拖拽中的条目半透明
- 悬停效果：浅灰背景

### 18.2 下次复习标签
- **今天**：红色标签（urgent）
- **明天**：灰色标签
- **N 天后**：灰色标签
- **具体日期**：灰色标签

### 18.3 时间格式化
- 刚刚 / N 分钟前 / N 小时前 / 月 日（相对时间）
- 列表和复习日志中都使用友好的中文时间格式

### 18.4 Toast 提示
- 底部居中悬浮提示，0.5 秒自动消失
- 从下方淡入动画
- 用于操作结果反馈（保存成功、删除成功、导出成功等）
- 500ms 内连续触发时覆盖前一条（不堆叠）

### 18.5 拖拽排序
- 自定义排序模式下，侧边栏条目可拖拽重排
- 拖拽时显示放置指示线
- 拖拽结束后持久化排序

### 18.6 鼠标滚轮导航
- 在编辑模式的主区域滚动鼠标滚轮可快速切换上/下一条错题
- 250ms 节流防止过快切换
- 如果目标元素内部有可滚动区域且未到达边界，不触发切换

### 18.7 Electron 窗口拖拽
- 工具栏区域设置 `-webkit-app-region: drag` 支持窗口拖动
- 按钮设置 `-webkit-app-region: no-drag` 保持可点击

### 18.8 题目区域图片工具
- 仅在非画笔模式下显示
- 鼠标悬停时渐显（`opacity-0 → opacity-100`）
- 包含拍照、桌面截图和导入图片三个按钮

### 18.9 摄像头拍照
- 调用 `navigator.mediaDevices.getUserMedia` 获取摄像头权限
- 前置摄像头优先（`facingMode: 'user'`）
- 视频预览 4:3 比例
- 拍照输出 JPEG（质量 0.9）的 DataURL
- 拍摄后自动插入光标位置或追加到编辑器末尾
- 关闭时自动释放摄像头资源

### 18.10 桌面截图（仅 Electron）
- 通过 `desktopCapturer` API 获取桌面窗口和屏幕的缩略图列表
- 3 列网格展示所有可截取的窗口和屏幕
- 选择截图后可交互式裁剪（拖拽四角调整裁剪区域）
- 裁剪后输出裁剪区域的 DataURL 并插入编辑器
- 支持 Enter 确认裁剪、Escape 取消

### 18.11 图像裁剪器
- 独立的交互式裁剪组件（配合桌面截图使用）
- 拖拽四角手柄调整裁剪区域
- 支持移动模式和创建模式切换
- 裁剪结果以 DataURL 输出

### 18.12 侧边栏折叠
- 侧边栏可折叠至 48px 宽（仅显示图标）
- 折叠状态下图标可触发返回、复习、统计、设置等核心操作
- 展开时宽度为 280px

### 18.13 工具栏紧凑模式
- 使用 ResizeObserver 监听工具栏宽度
- 宽度 < 680px 时自动切换为紧凑模式（隐藏部分文字标签）

## 19. 安全与消毒

### 19.1 HTML 内容消毒
- 复习模式下展示题目和答案内容时，使用 DOMPurify 对 HTML 进行安全清洗
- 防止通过粘贴内容注入 XSS 攻击脚本

### 19.2 Markdown 安全渲染
- 使用 `marked` 库解析 Markdown 内容
- 解析后通过 DOMPurify 进一步清洗，确保输出安全

## 20. PWA 离线支持

### 20.1 Service Worker
- 注册 Service Worker（`sw.js`），使用缓存优先策略
- 缓存名称 `cuoti-v2`
- 支持离线访问已缓存的资源

### 20.2 渐进式 Web 应用
- 应用可作为 PWA 安装到桌面
- 离线环境下仍可加载已缓存的应用外壳和数据
